# Metodologia de Neurovendas

Coloque nesta pasta os arquivos com a metodologia de neurovendas e perfil comportamental.

## Formatos suportados
- `.txt` (texto puro)
- `.md` (Markdown)

## Como funciona
Os arquivos são automaticamente carregados e injetados nos prompts de IA de análise de Neurovendas (consultas e ligações CRC). A IA usará **exclusivamente** o conteúdo destes documentos como base de conhecimento.

## Instruções
1. Coloque os arquivos de metodologia nesta pasta
2. Reinicie o servidor para atualizar o cache
3. Toda análise de Neurovendas será baseada nesses documentos

## Importante
- **NÃO commite** arquivos de metodologia proprietária no git (estão no .gitignore)
- O `README.md` é o único arquivo versionado nesta pasta
- Para atualizar a metodologia: substitua os arquivos e reinicie o servidor
